**Why:** Matches the existing `hubspot-integration.md` convention already in the repo.

**How to apply:** When creating or updating docs, write in Spanish. Use the same structure: intro paragraph, "Qué hace", "Archivos involucrados" table, gotchas, code snippets, and re-enablement/rehab instructions where applicable.
